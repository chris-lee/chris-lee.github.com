How I Built This Blog

Dec 1st, 2012

Here’s a quick overview of what went into this blog design.

Platform

The previous version of anotherchrislee (which is still available here) was built on Posterous which was a great platform in that it allowed me to write blogs by emailing them to Posterous, but it was pretty lacking in it’s ability to customize the design. This new version is built using the Octopress framework by Brandon Mathis. It’s described as a “blogging framework for hackers” so if you’re looking for something quick and dirty I’d suggest going with something else like Squarespace or Tumblr. However, if you’re willing to get your hands dirty in HTML and SASS and know how to self-host a site then I would definitely recommend Octopress. It’s been a lot of fun tweaking every detail to my heart’s content.

Another benefit to Octopress is that all posts are written in Markdown which provides a lot more flexibility with designing your posts compared to standard web editors.

Hosting

Octopress has some really helpful guides for hosting your blog and I ended up going with Github Pages since I love using git to manage projects and I wanted to give their pages tool a whirl. Two caveats to be aware of: if you have multiple github accounts (like one for work and one for personal stuff) it’s a little bit of a pain to setup multiple keys for deploying to git and you have to name your github repo username.github.com. I left off the “github.com” part initially which led to a little confusion.
